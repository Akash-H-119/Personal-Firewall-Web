Personal Firewall - Quick Run

Prerequisites:
- Python 3.8+
- pip install -r requirements.txt
- On Windows: install Npcap (https://nmap.org/npcap/)
- On Linux: iptables is available, and run as root for iptables changes.

1) Install dependencies:
   pip install -r requirements.txt

2) Edit rules.json to add your rules (or use CLI to add/remove).

3) Run CLI mode (sniffing + logging):
   python firewall.py

4) Run GUI mode (Tkinter):
   python firewall.py --gui

5) List rules:
   python firewall.py --list

6) Add a rule (example):
   python firewall.py --add block 192.168.1.5 22 TCP

7) Remove rule (by id):
   python firewall.py --remove <rule_id>

Notes:
- To enable system-level blocking automatically, allow the script to call iptables (Linux) or netsh (Windows). That requires Administrator/root privileges.
- Logs are written to firewall.log.
